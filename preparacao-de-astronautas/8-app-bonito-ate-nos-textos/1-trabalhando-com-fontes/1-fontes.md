### Basic Font Properties

* font-family
* font-weight
* font-style
* font-size

----------------------------------------------------------------

## Font Family

* Tipo de fonte de um elemento
* Lista de fontes e ordem de prioridade
* Inclui *fallback* font

```css
p {
  font-family: "Times New Roman", Times, serif;
}
```

  - serif
  - sans

----------------------------------------------------------------

## Font Weigth

* Peso da fonte

```css
p {
  font-weight: bold;
}
```

----------------------------------------------------------------

## Font Style

* O Estilo da fonte

```css
span {
  font-style: italic;
}
```

----------------------------------------------------------------

## Font Size

* O tamanho da fonte

```css
p {
  font-size: 18px;
}
```
